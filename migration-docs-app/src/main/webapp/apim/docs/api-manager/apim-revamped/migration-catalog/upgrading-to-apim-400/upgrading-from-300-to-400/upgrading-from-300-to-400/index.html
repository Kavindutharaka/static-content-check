    Upgrading from API-M 3.0.0 to 4.0.0¶

    To migrate from API-M 3.0.0 to 4.0.0, follow these steps:
    - Step 1: Perform the Configuration Migration.
    - Step 2: Execute the Extensions and Customizations Migration.
    - Step 3: Complete the Resource and Artifact Migration.
    - Step 4: Proceed with the Data Migration.
    - Step 5: Follow the instructions in Server Startup Actions to start the server.
